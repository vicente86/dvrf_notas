# ANOTA  AÍ

### Tecnologia

"Anota aí", é uma aplicação inicialmente feita com tecnologia web: html, css, js e reactjs.

### Descrição

A aplicação possibilita a criação, exclusão, o carregamento de notas.  

### Novas funcionalidades

* salvar as pasta em uma arquivo compactado
* Puxar um arquivo salvo
